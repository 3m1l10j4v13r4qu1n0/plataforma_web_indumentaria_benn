# Actores del Sistema
1. **Vendedor**: Inicia ventas, consulta stock en tiempo real y solicita cambios de productos.
2. **Cajero**: Finaliza la venta, genera el ticket/comprobante y procesa las devoluciones/cambios validando fechas y tickets.
3. **Gerente**: Autoriza excepciones, como descuentos que superan el porcentaje permitido.
4. **Encargado de Ventas**: Supervisa que las operaciones de venta y devolución actualicen el inventario automáticamente.
5. **Cliente**: (Actor externo) Solicita cambios y recibe el ticket de compra.
