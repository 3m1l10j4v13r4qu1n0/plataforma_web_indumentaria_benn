# Visión del Proyecto
Modernizar el proceso de ventas y post-venta de la empresa retail, eliminando la dependencia de procesos manuales (como consultar stock en depósito o actualizar inventario a mano). El objetivo es reducir pérdidas económicas por descuentos no autorizados o cambios de productos usados, garantizando una experiencia ágil para el vendedor, el cajero y el cliente.
