Historia de Usuario N°8
**Como** encargado de ventas,
**quiero** que el stock se actualice automáticamente,
**para** mantener información correcta del inventario.

Tarjeta
El sistema debe modificar el stock después de cada operación. Actualmente la actualización manual provoca diferencias en el inventario, por lo que se busca automatizar el proceso.

Criterios de Aceptación
- Dado que se realiza una venta de productos, cuando la operación se confirma, entonces el sistema debe descontar automáticamente la cantidad vendida del stock.
- Dado que un cliente devuelve un producto, cuando la devolución es aprobada, entonces el sistema debe incrementar el stock con los productos devueltos.
- Dado que se ejecuta cualquier operación de venta o devolución, cuando finaliza el proceso, entonces los cambios deben verse reflejados inmediatamente en el inventario.

Specification by Example
- Venta de 1 producto = stock disminuye en 1
- Devolución aprobada = stock aumenta en 1

Acceptance TDD
- Validar descuento de stock al vender (caso positivo)
- Validar incremento por devolución y actualización en tiempo real (caso negativo/borde)
