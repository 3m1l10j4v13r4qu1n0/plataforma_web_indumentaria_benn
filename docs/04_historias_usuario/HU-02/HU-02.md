Historia de Usuario N°2
**Como** cajero,
**quiero** registrar cambios de productos vendidos,
**para** cumplir con las políticas del negocio.

Tarjeta
Los clientes podrán cambiar productos dentro del plazo permitido de 15 días después de la compra. Fuera de ese tiempo no deben permitirse.

Criterios de Aceptación
- Dado que un cliente solicita un cambio de producto, cuando el cajero ingresa la fecha de compra, entonces el sistema debe validar la fecha de compra.
- Dado que la compra se realizó hace 15 días o menos, cuando el cajero procesa el cambio, entonces el sistema debe permitir el cambio.
- Dado que la compra se realizó hace más de 15 días, cuando el cajero intenta procesar el cambio, entonces el sistema debe informar que el plazo está vencido.

Specification by Example
- Compra hace 10 días = cambio permitido
- Compra hace 20 días = cambio rechazado

Acceptance TDD
- Probar cambio dentro del período válido
- Verificar mensaje de plazo vencido
