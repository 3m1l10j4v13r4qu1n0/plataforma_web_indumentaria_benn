Historia de Usuario N°5
**Como** gerente,
**quiero** limitar los descuentos realizados por vendedores,
**para** evitar pérdidas económicas.

Tarjeta
Los descuentos deben tener restricciones según el porcentaje permitido. El sistema debe controlar esos límites porque algunos vendedores aplican descuentos demasiado altos.

Criterios de Aceptación
- Dado que un vendedor aplica un descuento, cuando el porcentaje está dentro del límite permitido, entonces el sistema debe validar y aprobar el descuento.
- Dado que un vendedor intenta aplicar un descuento alto, cuando supera el porcentaje permitido, entonces el sistema debe requerir autorización.
- Dado que un descuento requiere autorización, cuando un gerente lo aprueba, entonces el sistema debe registrar quién autorizó el descuento.

Specification by Example
- Descuento 10% = aprobado
- Descuento 40% = requiere autorización

Acceptance TDD
- Validar porcentaje permitido
- Verificar solicitud de autorización y registrar usuario autorizador
